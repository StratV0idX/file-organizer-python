# 📂 File Organizer Script

A simple Python script to automatically organize files in a folder by type.

## 🚀 Features
- Organizes files into categories:
  - Images
  - Documents
  - Videos
  - Audios
  - Archives
- Automatically creates folders
- Easy to use

## 🛠️ How It Works
The script scans a folder and moves files into different folders based on their extensions.

## 📦 Requirements
- Python 3.x

## ▶️ How to Use
1. Download or clone this repository
2. Run the script
3. Enter your folder path:
4. End
